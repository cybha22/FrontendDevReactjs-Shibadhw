# FrontendDevReactjs - Shbdhw

## Tech Stack
- React.js v19.2.4
- React Router DOM v7.14.0
- Vite v8.0.4
- Node.js v22.16.0

##  Start Project

### npm
```
npm install
npm run dev
```

### yarn
```
yarn install
yarn dev
```

## Login
no
## Build for Production
```
npm run build
npm run preview
```

## Architecture
```
src/
  services/       -> Lapisan layanan API (ambil data asinkron)
  hooks/          -> Hook kustom (useRestaurants, useRestaurantDetail)
  components/     -> Komponen UI yg dapat digunakan kembali 
  pages/          -> Page level components
```
